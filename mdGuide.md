## Markdown description

Sobre os arquivos md, há um guia bem intuitivo que mostra, na prática, sua utilização como uma linguagem de marcação.
O [Dillinger] é um editor web com preview, indicado, inclusive, pela organização oficial [MarkdownORG]

O editor disponibiliza o que seria uma documentação da linguagem, salva nessa pasta em um arquivo próprio (mdGuideExample.md).

[Dillinger]: <https://dillinger.io/>
[MarkdownORG]: <https://www.markdownguide.org/>
